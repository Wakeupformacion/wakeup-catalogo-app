# Branding base aplicado

## Referencia visual
Se ha tomado como base el logo `WakeUpLogo.png`.

## Rasgos detectados
- color principal: rojo-anaranjado enérgico
- color secundario: gris carbón / oscuro
- estilo: moderno, geométrico, tecnológico pero accesible

## Decisiones visuales trasladadas a la demo
- CTA principal en rojo-anaranjado
- fondos claros con contraste alto
- tipografía sans limpia
- tarjetas con esquinas redondeadas
- jerarquía visual clara para búsqueda y catálogo

## Objetivo
Transmitir una sensación de:
- formación online moderna
- descubrimiento sencillo
- tecnología útil
- catálogo amplio pero ordenado

## Siguiente evolución sugerida
- afinar paleta exacta a partir de guía de marca real
- añadir iconografía coherente
- incorporar métricas visuales, badges y microcopys comerciales
